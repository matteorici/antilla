# Code Snippets Manager

A lightweight application for organizing reusable code snippets.

The project stores snippets in memory, supports searching by programming language and tags, and prints formatted results.

---

## Example

```
Code Snippets

Python

read_json

Tags:
json file parsing

-------------------------

JavaScript

debounce

Tags:
events performance
```

---

## Modules

- snippet.py — snippet model
- library.py — snippet collection
- search.py — filtering
- repository.py — data source
- printer.py — output
- sample_data.py — demo snippets

Run

```
python app.py
```

Uses sample data only.
